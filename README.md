# NifScan
.nif (GameBryo Format File) file parser

Парсер игровых файлов .nif для игры Skyrim. Возможность массово производить изменения в файлах. Возможность полной выдачи debug информации по каждому файлу, возможность смещать vertex, массово перекрашивать\очищать vertex цвета, изменять эмиссию цвета, изменять флаги шейдеров, пересортировывать блоки, получать предупреждения о неверных параметрах модели и автоматически исправлять некоторые распространенные ошибки, добавлять\удалять параллакс для определенных текстур, заменять коллизию, высокая производительность. Программа легко перенастраивается под любые запросы благодаря полному парсу заголовка и всего строения файлов.
